# TaskTracker
c++ app for task tracking

# Requirements
- CMake 3.10 or higher
- vcpkg

# Building
- clone repository  
`git clone https://github.com/DanyloKalinchuk/TaskTracker`  
`cd TaskTracker`

- build with CMake  
`mkdir build`  
`cd build`  
`cmake ..`  
`make`

# Usage
While the build folder use `.\TaslTracker` app name.  
Print `.\Tasktracker help` to see the list of commands.
